# Data Directory

This directory contains:

- `supermarkets-cache.json` - Product database cache (auto-downloaded)
- `grocery-feedback.jsonl` - Your price feedback data (local only)
- `receipts/` - Receipt images (never commit these!)

## Privacy Note

Never commit:
- Real receipt images
- Personal grocery feedback data  
- Any files containing personal shopping information

The system keeps all your data local and private.
